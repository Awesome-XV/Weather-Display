# Weather-Display

# Why
I made this project as a fun way to check the weather whenever you power it on, and if you add functionality it could become worth far more than just a way to check the weather.

# What
This is a project that uses an ESP32 to check the weather at a location that you specify, it will pull from a free API, you will have to provide the API key. And it will provide info about that location and you can even scroll past screens so know more about it. There are 3 switches that are incorporated to allow you to scroll past screens. Then there is the functionality of adding other ways to use it like checking specific things for you.
